## How to build
1. Make sure Haxe and HaxeFlixel are installed. This includes Flixel Addons.
2. Should be able to compile it with a "lime test (windows/mac/linux) -debug"
3. If you can't, I messed up something bad and you should make an issue about it. Or maybe you installed Haxe and HaxeFlixel wrong. Make sure you have the latest versions! This includes Flixel Addons.
